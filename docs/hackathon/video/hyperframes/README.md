## Render

Each project renders to `raw/hf_<name>.mp4` under `../raw/`.

```bash
# One-off
cd vigia
npx hyperframes lint
npx hyperframes render --output ../../raw/hf_vigia.mp4

# All four
for p in vigia evidence_pack reasoning_chain sinagir_export; do
  (cd "$p" && npx hyperframes render --output "../../raw/hf_${p}.mp4")
done
```

Quality flags:

- `--quality high --fps 30` for final.
- `--quality draft` while iterating.
- `--strict` to fail on lint errors.

## Regenerate after editing `ui_scenes/*.html`

```bash
python3 ../scripts/build_hyperframes.py
python3 ../scripts/check_hyperframes.py     # asserts timeline+layout
```

## Constraints honored

- Standalone compositions: root `<div data-composition-id>` lives directly in
  `<body>` (no `<template>` wrapper).
- Every timeline registers via `window.__timelines["<id>"] = tl`.
- Timeline starts paused; framework owns playback.
- `data-duration` set on the root composition so HF uses configured duration.
- Final tween pads the timeline so visual hold matches duration.
- No `repeat: -1`, no `Math.random()`, no async timeline construction.
